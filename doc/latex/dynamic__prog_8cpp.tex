\doxysubsection{Detailed Description}
Implementação do algoritmo de programação dinâmica para resolver o problema da mochila. 

Este arquivo contém a implementação do algoritmo de programação dinâmica para resolver o problema da mochila, que é uma variação do Problema da Mochila 0/1. O algoritmo utiliza uma abordagem bottom-\/up para construir uma tabela onde dp\mbox{[}i\mbox{]}\mbox{[}w\mbox{]} representa o lucro máximo que pode ser obtido com os primeiros i paletes e uma capacidade de peso w.

\begin{DoxyNote}{Note}
Complexidade de Tempo\+: O(n \texorpdfstring{$\ast$}{*} W), onde n é o número de paletes e W é a capacidade do caminhão. Isso ocorre porque o algoritmo preenche uma tabela de dimensões (n+1) x (W+1). 

Complexidade de Espaço\+: O(n \texorpdfstring{$\ast$}{*} W), devido à tabela dp de dimensões (n+1) x (W+1). 
\end{DoxyNote}

Solves the 0/1 Knapsack Problem using dynamic programming approach. 

This function implements a bottom-\/up dynamic programming solution to the Delivery Truck \doxylink{structPallet}{Pallet} Packing Optimization Problem, which is a variation of the 0/1 Knapsack Problem. The algorithm creates a 2D table where dp\mbox{[}i\mbox{]}\mbox{[}w\mbox{]} represents the maximum profit that can be obtained with the first i pallets and a weight capacity of w.

For each pallet i and capacity w, the algorithm decides whether to include the pallet in the solution by comparing\+:
\begin{DoxyEnumerate}
\item The profit from not including the pallet (dp\mbox{[}i-\/1\mbox{]}\mbox{[}w\mbox{]})
\item The profit from including the pallet (dp\mbox{[}i-\/1\mbox{]}\mbox{[}w-\/wt\mbox{]} + pf)
\end{DoxyEnumerate}

After building the table, the algorithm traces back the selected pallets and outputs the total profit, total weight, and the IDs of the selected pallets.
