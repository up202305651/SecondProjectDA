Este projeto foi desenvolvido com o objetivo de criar uma ferramenta de otimização de empacotamento de paletes que permite aos usuários selecionar diferentes conjuntos de dados e algoritmos para resolver o problema da mochila (Knapsack Problem). A ferramenta é acessível através de um menu de linha de comando e foi projetada para ser intuitiva e eficiente.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Objetivos}{Objetivos}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Leitura e Processamento de Dados}}\+: Implementar funcionalidades para ler e processar arquivos de dados, armazenando-\/os em estruturas de dados apropriadas.
\item {\bfseries{Seleção de Algoritmos}}\+: Oferecer uma variedade de algoritmos para resolver o problema da mochila, incluindo Força Bruta, Programação Dinâmica, Abordagem Gulosa e Programação Linear Inteira.
\item {\bfseries{Exibição de Resultados}}\+: Apresentar os resultados de maneira clara e direta, permitindo uma avaliação eficaz das diferentes abordagens.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Funcionalidades Principais}{Funcionalidades Principais}}\label{index_autotoc_md3}

\begin{DoxyItemize}
\item {\bfseries{Menu de Linha de Comando}}\+: Permite aos usuários selecionar conjuntos de dados e algoritmos de maneira fácil e intuitiva.
\item {\bfseries{Leitura de Arquivos CSV}}\+: Funcionalidades robustas para ler e processar arquivos CSV contendo dados de paletes e caminhões.
\item {\bfseries{Algoritmos de Otimização}}\+: Implementação de múltiplos algoritmos para resolver o problema da mochila, cada um com suas próprias características e vantagens.
\item {\bfseries{Estruturas de Dados Eficientes}}\+: Utilização de vetores e estruturas personalizadas para armazenar e manipular dados de maneira eficiente.
