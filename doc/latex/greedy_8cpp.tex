\doxysubsection{Detailed Description}
Implementação do algoritmo guloso para resolver o problema da mochila. 

Este arquivo contém a implementação do algoritmo guloso para resolver o problema da mochila, ordenando os paletes pela razão lucro/peso. O algoritmo guloso é uma abordagem heurística que seleciona localmente a melhor opção em cada etapa, com a esperança de encontrar uma solução globalmente ótima.

\begin{DoxyNote}{Note}
Complexidade de Tempo\+: O(n log n), devido à ordenação dos paletes pela razão lucro/peso. A ordenação é feita uma vez, e depois o algoritmo itera sobre os paletes uma vez. 

Complexidade de Espaço\+: O(n), onde n é o número de paletes. O espaço adicional é usado principalmente para armazenar a solução atual. 
\end{DoxyNote}
